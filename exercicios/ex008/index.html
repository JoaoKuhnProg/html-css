Textos Marcados

Podemos criar tambem um texto marcado usando a tag "mark"(Semantica).

Estamos criando um Texto grande e um texto pequeno nesse paragrafo, respectivamente sendo as tag "big"(Não semantica) e "small"(Semantica).

Texto Deletado

Podemos marcar um texto como excluido, para identificar que elee deve ser lido, mas não considerado, com a tag "del"(Semantica).

Texto Inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois, usando a tag "ins"(Semantica).

Existe tambem o sublinhado, com a tag "u"(Não semantica).

Texto sobrescrito

Para inserir coisas do tipo x2+3, é utilizado a tag "su"(Semantica).

Texto subscrito

Para inserir coisas do tipo H20, é utilizado a tag "sub"(Semantica).
